# Quanta

###### Collaborative Documents, Wikis, Social Media, ChatGPT, Fediverse, Semantic Web

<img src='attachments/5f786b72982c4018afcb57fd-eagle-logo-250px-tr.jpg' style='width:20%'/>


**Quanta** is an [Open Source](https://github.com/Clay-Ferguson/quantizr) `Hierarchical Content Management Platform`. Content text, images, and other file attachments can be organized on a hierarchical tree and edited, published, shared, etc. Quanta uses the `"block editor" approach` to document editing, which means individual blocks of content (called `Nodes`) make up all of the data.

Platforms using block editors are becoming very popular, and include for example:  [Notion](https://notion.so), [Jupyter Notebooks](https://jupyter.org), and [WordPress Gutenberg](https://wordpress.org/gutenberg).
