# Quanta

###### Collaborative Documents, Wikis, Social Media, ChatGPT, Fediverse, Semantic Web

<img src='attachments/5f786b72982c4018afcb57fd-eagle-logo-250px-tr.jpg' style='width:20%'/>


**Quanta** is an [Open Source](https://github.com/Clay-Ferguson/quantizr) `Hierarchical Content Management Platform`. Content text, images, and other file attachments can be organized on a hierarchical tree and edited, published, shared, etc. Quanta uses the `"block editor" approach` to document editing, which means individual blocks of content (called `Nodes`) make up all of the data. Platforms using block editors are becoming very popular, and include for example:  Notion, Jupyter Notebooks, and WordPress Gutenberg.

# Get Started

Quanta is `Fediverse-enabled (aka Mastodon)`, so you can follow people and/or post messages. Try it now by viewing the [Public Feed](https://quanta.wiki?view=feed) or the [Trending Topics](https://quanta.wiki?view=trending), or else try out the RSS Reader with this subscription of [Random RSS Feeds](https://quanta.wiki/u/wclayf/random-rss)
